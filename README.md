Описание
Windows Forms приложение для автоматизации работы транспортной компании с интеллектуальной системой тарифов и управлением заказами.

Основные функции
 ✅ Управление тарифами - создание, редактирование, категоризация тарифных планов
 ✅ 4 типа грузов - общие, хрупкие, опасные, скоропортящиеся
 ✅ Гибкая система скидок - настраиваемые скидки от 0% до 100%
 ✅ Автоматический расчёт - мгновенный расчет стоимости перевозок
 ✅ Управление клиентами - база клиентов с контактной информацией
 ✅ Обработка заказов - полный цикл от создания до расчета
 ✅ Валидация данных - проверка корректности всех вводимых значений
 ✅ Локализованный интерфейс - полностью на русском языке
