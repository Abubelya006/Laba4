Описание
Windows Forms приложение для автоматизации работы транспортной компании с интеллектуальной системой тарифов и управлением заказами.

Основные функции
1. ✅ Управление тарифами - создание, редактирование, категоризация тарифных планов
2. ✅ 4 типа грузов - общие, хрупкие, опасные, скоропортящиеся
3. ✅ Гибкая система скидок - настраиваемые скидки от 0% до 100%
4. ✅ Автоматический расчёт - мгновенный расчет стоимости перевозок
5. ✅ Управление клиентами - база клиентов с контактной информацией
6. ✅ Обработка заказов - полный цикл от создания до расчета
7. ✅ Валидация данных - проверка корректности всех вводимых значений
8. ✅ Локализованный интерфейс - полностью на русском языке
